Question #2 - JavaScript

Ajouter un titre dynamiquement

Lorsqu'on clique sur le bouton:

  • La balise qui s'ajoute est un h3
  • Le texte du titre doit être un nom aélatoire entre Jean-François, Jean-Sébastien, Marie-France et Matthieu(indice Tableau)
  • Le titre doit être ajouté dans le div .title_holder
  • Doit apparaitre dans une position alétoire sur l'axe horizontal et vertical
  • Vous pouvez ajouter un identifiant sur votre bouton dans le html

Question #3 - CSS

Le responsive

Vous devez faire la mise en page en colonne qui s’adapte au différente résolution

  • 4 colonnes par défaut
  • 2 colonnes lorsque la résolution est de 768
  • 1 colonne lorsque la résolution est de 375
  • N'oubliez pas que nos mises en page devraient être fonctionnelle peut importe la résolation
  • Vous devez faire la bonne gestion de l'espacement sur les côtés du site, lorsque la résolution est plus petite
  • Dans chacune des cartes il y a une image, vous devez choisir le bon format(il y a deja les 3 formats dans le dossier assets)
